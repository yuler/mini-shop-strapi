# mini shop

> 微信小程序

## Features

- Home page for list products
- API power by [strapi](https://strapi.io/) host on (https://strapi.ganfans.com)

## Development

```bash
cp .env.example .env
pnpm install
pnpm run ide:npm
pnpm run ide:open
pnpm run dev
```
